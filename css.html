Merhabalar öncelikle ben osman wissen akademie sürekli eğitim merkezinde web ve mobil programlama bölümü kursiyerim bu gün sizlere flex yapısı hakkındaki bilgilerimi paylaşacağım. Öncelikle flex yapusı bizim nerede işimize yarar? Bu sorunun cevabı oldukça basit biz sitemizin ön yüzünde tag(nesne)lerimizi yerleştirme yaparken bize esneklik sağlar. Esneklikten kastım şudur: orneğin div tagı bir blok elementtir blog elementler verilen width ve height değerleri ne olursa olsun bir sonraki tag ı alt satıra atmaktadır fakat flex yapısını kullanarak bu olaydan kurtulabiliyoruz nasıl mı?

Örnek: html kodlarımıza gelerek 1 div bloğumuzun içine 3 adet div bloğu oluşturalım ve alt tarafta olduğu gibi css verelim

Evet şuan bu html ve css kodlarımızı yazdığımıza göre çalıştırdığımızda bize alt resimde de gördüğümüz gibi alt alt a farklı renklerden oluşan divleri gösterecek.



Fakat biz bu divleri yan yana görmek istediğimiz bir projemiz olduğunu düşünürsek eğer işte bu kısımda flex yapısında faydalanıyoruzbu kısımda mainDiv css ime display:flex ekleyerek şunu demiş oluyoruz benim bütün nesnelerim inlineolacak yani blok da olsa inline da olsa html bu nesneleri inline algılayacak. Hemen deneyelim. Css dosyama gidip mainDiv css ime display:flex; i ekliyorum


Bakın gördüğünüz gibi artık nesnelerim yan yana geldi. Aslında display:flex in asıl amacı budur. Fakat eğer istersek biz hem display özelliğine flex verip yine de ben nesnelerimi alt alta kullanmak istiyorum derseniz ki bu bazen lazım oluyor bu durumda ise flex-direction: column; aracını kullanarak html benim main divim in içerisindeki nesnelerin display özelliği flex olsun fakat ben bunları yinede alt alta görmek istiyorum diyorsunuz aşşağıda olduğu gibi.



Şimdi diyeceksiniz ki ya ben zaten alt alta görüyordum nesnelerimi bu benim bir işime yaramadı bazı ince ayrıntılar var örneğin bizim kullandığımız nesneler block element nesnelerdi yani divler di bunun yerine biz block nesneler değilde inline nesneler kullansaydık html bunu bu nesneler inline olsa bile biz flex-direction: column; ı eklediğimiz için bunları blok nesne olarak görecekti evet ne dediğimi daha iyi anlamak için şimdi divlerimin yerine inline nesne olan span etiketini kullanarak 3 cümle yazacağım bakalım nasıl olacak.


     


Ust kısımda da gördüğümüz üzere flex-direction: column; özelliği sayesinde inline olan nesneleri block yaptık peki flex yapısı sadece bu kadar için mi yarıyor işimize tabikide hayır nesneleri daha düzgün yerletirmemize oldukça işe yarınyor bunlardan birisi justify-content özelliği eğer siz display:flex; özelliği verir ve flex-direction: column; yapmazsanız justify-content:center özelliği size yatay hizalamayı ortalayacaktır. Hadi deneyelim bu kısımda daha iyi anlamanız açısında ilk örneğimiz olan divler ile devam edeceğim ilk olarak css dosyamıza gidiyoruz ve mainDiv css imize justify-content:center özelliğini ekliyoruz



Evet gördüğümüz gibi justify-content:center özelliği bizim yan yana bulunan nesnelerimi ortaladı. ama display:flex özelliği bu kadar ile de sınırlı değil size hizalama alanında dahada falza seçnek suyor örnekler alt kısımda



1 numaralı görümü yapmak için justify-content:space-between özelliğini kullanıyoruz.
2 numaralı görümü yapmak için justify-content:space-around
3 numaralı görümü yapmak için ise justify-content:space-evenly özelliğini kullanıyoruz

Peki hocam bitti mi flex özelliği tabikide hayır pekala dikey de hizalama yapaken ne kullanıyoruz. align-items: center;
bu özellik ise size dikeyde ortalama sağalayacaktır.Hemen örneğimizde deneyelim ilk olarak mainDiv css imize align-items: center; özelliğini ekleyererk başlayalım.



Evet gördüğünüz gibi justify-content:center ve align-items:center özelliğini kullanarak nesnelerimi hem yatay eksende hem dikey eksende hizalamarını kolay bir şekilde yaptık ÖNEMLİ BİLGİ: Eğer bu iki özelliği flex-direction:column iken kullanır iseniz tam tersi bir hizlama yapacaktur yani biz justify-content i yatayda hizalamak için kullanırken flex-direction:column iken dikeyde hizalamak için kullanıyor olacağız aynı şekilde align-items özelliğinide de yatayda hizalama yapmak için kullanıyor olacağız.
